# Auto Attendance Form Extension

Google フォームの出席フォームを自動で入力・送信し、送信後はタブも自動で閉じるブラウザ拡張機能です。Chrome と Firefox の両方に対応しています。

ダウンロードはこちらから: [GitHub Releases](https://github.com/Narcissus-tazetta/AutoAttendanceForm/releases)

## 技術スタック（現状ファイルに基づく）

- TypeScript
- React
- Tailwind CSS
- esbuild
- WXT
- bun

# 概要

この拡張機能は、Google フォームの出席フォームを自動で入力・送信し、送信完了後にタブを自動で閉じることで、繰り返しの出席登録を効率化することを目的としています。Chrome と Firefox に対応し、設定可能なフィールドマッピングや遅延オプションを備えています。

# 特徴

- 自動入力：指定したフィールドに自動で値を入力します（名前、メール、コメント等）。
- 自動送信：入力後にフォームを自動で送信します。
- 自動タブクローズ：送信完了後にタブを閉じて作業を自動化します。
- 設定可能：フィールドセレクタ、送信までの遅延、繰り返し回数などを拡張オプションで調整可能。
- クロスブラウザ対応：Chrome と Firefox の両方で動作確認済み。
- 軽量・ローカル処理：データは基本的にローカルで処理され、外部送信は行いません（設定により異なります）。

簡単な使い方

1. ビルドして（esbuild / bun）拡張をアンパックで読み込みます。<u>**bunを推奨**</u>
2. 対象のフォームを開き、拡張のオプションでフィールドマッピングと遅延を設定します。
3. 実行して自動送信を確認します。
